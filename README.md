## 😎 프로젝트 후기
> 주차 예약 시스템을 만들기 전에는 예약 이라는 것에 대해 가볍게 생각하고 있었습니다.
> 하지만 그것은 큰 오차였던것 같습니다. 실제로 예약 기능과 검색기능, 회원가입등
> 그동안 쉽고 간단하게 사용하던 모든것들이 많은 개발자분들의 땀과 노력이 들어갔다는 것을 이번일로
> 많이 느끼고 깨달았던 것 같습니다. 그리고 개인 프로젝트를 진행하면서 저 또한 많이 발전하고 성장한 것 같습니다.
> 비록 팀원없이 혼자 한 프로젝트 이지만 처음부터 끝까지 책임감을 가지고 만들게 된 것 같습니다.
> 다음에 기회가 된다면 팀프로젝트로 더 좋은 프로젝트를 만들어 보고 싶습니다.
